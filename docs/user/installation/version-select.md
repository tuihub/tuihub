## 版本语义

librarian（服务端） 和 waiter（客户端）的版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。  

:::warning
> 语义化版本控制规范  
> 4. 主版本号为零（0.y.z）的软件处于开发初始阶段，一切都可能随时被改变。这样的公共 API 不应该被视为稳定版。

目前双端主版本号均为零，只有按照下列推荐的组合使用才能获得最大的兼容性。
:::
